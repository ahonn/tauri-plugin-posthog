## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-capture`
- `allow-identify`
- `allow-alias`
- `allow-reset`
- `allow-get-distinct-id`
- `allow-get-device-id`
- `allow-get-effective-distinct-id`
- `allow-is-auto-identify-enabled`
- `allow-capture-batch`
